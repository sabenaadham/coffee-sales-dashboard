# Coffee Sales Dashboard

## Project Overview
This project is an Excel-based sales dashboard created to analyze coffee sales performance and generate business insights.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning
- Dashboard Design

## Key Insights
- Identified top-performing products
- Analyzed regional sales trends
- Monitored monthly revenue performance
- Visualized customer purchasing behavior

## Features
- Interactive dashboard
- KPI tracking
- Sales trend analysis
- Dynamic filtering

## Project Outcome
This dashboard helps businesses make data-driven decisions by analyzing sales patterns and performance metrics.
